## 🌡️ Climat — Pourquoi Tenerife Sud en février ?

| Critère | Tenerife Sud | Notes |
|:--------|:------------:|:------|
| **Température moyenne** | **22-25°C** | Le plus chaud des Canaries en hiver |
| **Eau de mer** | ~20°C | Fraîche mais agréable pour barboter |
| **Ensoleillement** | **7h/jour** | Temps sec, peu de pluie |
| **Vent** | **Faible** | Protégé des alizés par le Teide |
| **Humidité** | Modérée | Très agréable |

> 💡 Le **microclimat du sud** (Costa Adeje) est bien plus favorable que le nord (brumeux et humide en février). C'est la meilleure île des Canaries pour cette période.

## 👶 Checklist pratique — Voyager avec Célestine (11 mois)

| Point | Détail |
|:------|:-------|
| ✈️ **Billet bébé** | Lap infant = sur les genoux, pas de siège dédié (Ryanair ~30-50€ vol, TUI Fly gratuit/taxes) |
| 🛄 **Poussette** | Prendre une poussette compacte cabine ou un porte-bébé |
| 🛏️ **Lit bébé** | Demander à l'hôtel (gratuit, à réserver à l'avance) |
| 🍼 **Repas** | All-inclusive = buffet avec purées, légumes vapeur, fruits. Prévoir quelques petits pots de sécurité |
| 🌡️ **Piscine** | Privilégier les hôtels avec piscine chauffée (tous les 4 ci-dessus l'ont) |
| 🧴 **Crème solaire** | SPF50 bébé — le soleil canarien est fort même en février |
| 👕 **Vêtements** | Leggings/t-shirts le jour, petite veste/ pull le soir (brise possible) |
| 💊 **Pharmacie** | Paracétamol bébé, thermomètre, crème change, sérum physiologique |
| 🚖 **Transports** | Siège auto bébé pour le transfert aéroport (à demander au taxi/transfert) |
