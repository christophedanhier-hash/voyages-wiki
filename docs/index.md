## 🔄 Extraction automobile

Ce wiki est alimenté par le script `polarsteps_extract.py` qui interroge l'API Polarsteps.
Pour extraire un nouveau voyage terminé :

```
echo 'ton_token' | /opt/hermes/.venv/bin/python3 /opt/data/polarsteps_extract.py
```

*Données extraites depuis [Polarsteps](https://www.polarsteps.com/christophedanhier/)*
